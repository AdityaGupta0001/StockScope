About StockScope

This project demonstrates a modern, end-to-end CI/CD pipeline for deploying a containerized web application. The entire workflow is automated, from code commit to live deployment.

Key Technologies Used:

  • Python & Flask: Lightweight and robust backend server.
  • Docker: Containerization for consistent environments.
  • Jenkins: Automation server to build, test, and deploy.
  • Kubernetes (Minikube): Orchestration for deployments.
  • GitHub: Version control and pipeline triggers.

When a developer pushes a change to the GitHub repository, Jenkins automatically detects it, builds a new Docker image, and deploys the updated version to the Kubernetes cluster with zero downtime.
